Authentication
All protected routes require a JWT Bearer token in the Authorization header. Tokens are issued on signup and login.
POST
/api/auth/signup
Register a new user account
Public
Request Body
{
  "email": "user@example.com",
  "password": "Secure@123"
}
Password must be min 8 chars, include uppercase, a number, and a special character.
Response 201
{
  "user": {
    "_id": "6a37c7c2...",
    "email": "user@example.com",
    "role": "user",
    "skills": [],
    "avatarUrl": ""
  },
  "token": "eyJhbGci..."
}

Tickets
After a ticket is created, an Inngest background job automatically runs AI triage — assigning priority, helpful notes, and matching a moderator based on required skills.
POST
/api/tickets
Create a new support ticket and trigger AI triage
🔒 Auth
Request Body
{
  "title": "React app shows white screen",
  "description": "My React app crashes on load with no error. It was working yesterday."
}
Response 201
{
  "message": "Ticket created and processing started!",
  "ticket": {
    "_id": "6a37cfa4...",
    "title": "React app shows white screen",
    "status": "TODO",
    "priority": null,
    "createdBy": "6a37c7c2...",
    "createdAt": "2026-06-21T11:48:00.000Z"
  }
}
Priority and skills are populated asynchronously by the AI Inngest workflow after creation.

POST
/api/tickets/:id/responses
Add a reply to a ticket thread
🔒 Auth
Request Body
{
  "message": "I tried clearing the cache but the issue persists."
}
Ticket status updates automatically: user reply → TODO, moderator reply → IN_PROGRESS.
Response 200
{
  "success": true,
  "ticket": {
    "_id": "6a37cfa4...",
    "status": "TODO",
    "responses": [
      {
        "senderId": "6a37c7c2...",
        "senderRole": "user",
        "message": "I tried clearing the cache...",
        "createdAt": "2026-06-21T12:05:00.000Z"
      }
    ]
  }
}
POST
/api/tickets/:id/draft
Generate an AI-powered reply draft using Groq LLM
⭐ Mod/Admin
Response 200
{
  "draft": "Hello! Thank you for reaching out. I've reviewed your ticket regarding the white screen issue in your React application.\n\nBased on the AI triage notes, this is most likely caused by an incorrect usage of React hooks such as useEffect or useMemo.\n\nCould you please open your browser's developer console (F12) and share any error messages you see there? That will help me pinpoint the exact cause.\n\nBest regards, Support Team"
}
Uses Mixtral via Groq. The AI is guided by the helpfulNotes generated during the initial triage step.
